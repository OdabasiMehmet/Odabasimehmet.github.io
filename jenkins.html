Jenkins Pipeline for Web Page Application (Postgresql-Nodejs-React)


This project aimed to create a Jenkins pipeline to deploy a web-page application written with Nodejs and React Frameworks. AWS were used in terms of Cloud Infrastructure and created by using Ansible. The infrastructure had 1 jenkins server (`Amazon Linux 2 AMI`) as ansible control node and 3 EC2's as worker node (`Red Hat Enterprise Linux 8 with High Availability`). These EC2's were launched on AWS console. Web-page had 3 main components which are postgresql, nodejs, and react. Each component was serving in Docker containers on EC2s that were dedicated for them. Postgresql was serving as Database of the web-page. Nodejs controlled backend part of web-side and react controlled frontend side of web-page.
